핵심 : 예외(exception)처리
주로 람다, map 함수 같이 사용

strip() : 문자열의 양쪽 끝에 있는 공백이나 특정 문자를 제거

***예외 처리를 통해 실행 상태 유지 가능 - 정상종료

try:
	코드블록
except 예외클래스:
	코드블록
except 예외종류 as 변수명:
	코드블록
raise 예외종류("에러 메시지")


finally:
	코드블록			#무조건 처리 되는 코드

len() 함수 : 객체의 길이(요소 개수)를 정수(int)로 반환

ord(인수) : 문자를 정수형 숫자로 변환
ch(인수) : 숫자를 문자로 변환

형식지정자,(=서식문자)

f-string : 따옴표 한번만 써도 되게, 변수는 {} 처리
name = "세훈"
age = 29
print(f"안녕하세요, 제 이름은 {name}이고 나이는 {age}살입니다.")


포멧 메소드
text = "이름: {}, 나이: {}".format("세훈", 29)
print(text)

문자열 함수 3가지
***중요split('구분자') : 리스트로 반환, 구분자 없으면 공백을 구분자로 인식
strip() : 문자열 양끝 공백 제거
***'구분자'.join(리스트) : iterable(반복가능한), 결합


이스케이프(escape) 코드: 미리 정의해 둔 문자조합
\n = newline
\t = tap
\b = backspace
\' = 작은 따옴표 표시
\" = 큰 따옴표 표시
\r = 커서를 현재 줄의 가장 앞으로 이동

lambda 인자 : 표현식
ex lamda x, y : x+y
map(함수, 이터레이터) : iterable(반복가능한) 객체의 각 요소 함수 적용
지정한 iterable 데이터를 함수에 넣어 iterator로 반환
map(람다함수, 이터레이터) : 람다함수 자주 사용

dict.items() : 키(key)와 값(value)을 한 쌍으로 묶어서 돌려주는 메서드
ex)
student = {"name": "세훈", "age": 29, "major": "EE"}
print(student.items())
