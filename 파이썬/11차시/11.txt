tkinter 내장모듈, 하위(sub)모듈
위젯 : 작은 프로그램(객체)

라이브러리 = 라이브러리 모듈, 라이브러리 패키지
안에는 함수, 변수, 클래스

오늘의 핵심
위젯 클래스
1. 객체 생성 
2. 배치
3. 이벤트

객체 생성 이유
1. 기능(함수) 사용 2. 속성(변수) 사용

coomand =  함수명 으로 이벤트를 바인딩

함수(괄호없이 사용하는 경우)
1. 변수에 저장
2. 다른 함수에 전달
3. 반환값으로 사용 가능

get은 입력된 값을 가져오는 메서드
ex) value = radio_value.get() -> Intvar 객체 멤버 변수 값 가져오기

set은 값 저장
ex) radio_value.set(-1)   ->   Intvar 객체 멤버 변수 값 저장하기

Stringvar() (문자열)변수값 위젯과 연결
IntVar 정수 저장에 사용하는 클래스(멤버변수에 저장)

ex) variable=radio_value, value = 3) 3을 선택 하면 3을 저장하겠다.

Geometry manager : 배치 관리자
geometry('가로x세로+x좌표+y좌표')

palce()
