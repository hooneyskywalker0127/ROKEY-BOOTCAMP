re모듈(=정규표현식) regular expression

화이트스페이스 = 공백문자
\r = Carriage Return : 현재 줄 맨 앞으로 이동
\f = form feed : 다음페이지로 넘김
\v = vertical tab : 수직 탭
\s 화이트스페이스. 공백
\w 대소문자,숫자


컴파일 옵션
S 
I 
M ^$
X  \n, 주석

* 0부터 무한대 반복
+ 1부터 무한대 반복

표준 라이브러리 = 내장 라이브러리

match, search는 정규식과 매치될 때, match 객체를 리턴하고 매치되지 않을 때는 None을 리턴. 

match는 첫글자부터 검색

문법** 
match는 첫글자부터 검색
패턴 = re.compile("정규표현식")
매치객체 = 패턴객체.match("검색 대상 문자열")
print(매치객체)

패턴 = re.compile("정규표현식")
매치객체 = 패턴객체.search("검색 대상 문자열")
print(매치객체)
